## Project Overview

This is the RAGFlow official documentation site (`ragflow.io`), built with [Docusaurus 3.9.2](https://docusaurus.io/). The docs are auto-synced from the main RAGFlow repo at `github.com/infiniflow/ragflow`.

## Common Commands

```bash
cd website

npm                  # Install dependencies
npm start             # Local dev server (http://localhost:3000)
npm build             # Build for production
npm deploy            # Deploy to GitHub pages
npm typecheck         # TypeScript type checking
npm clear             # Clear Docusaurus cache
npm genversions       # Regenerate versioned docs from versions.json
npm validateplugin    # Run plugin validation
```

## Architecture

### Documentation Structure

- `website/docs/` — Main documentation (guides, references, API docs, FAQ)
- `website/basics/` — Foundational concepts (separate docs instance at `/basics`)
- `website/blog/` — Blog posts with yearly-organized sidebar
- `website/versioned_docs/` — Historical versioned documentation (v0.10.0 through v0.24.0)

### Docusaurus Configuration

- `docusaurus.config.ts` — Main Docusaurus config; defines two doc instances:
  - **default**: docs at `/docs` route
  - **basics**: separate instance at `/basics` route
- `sidebars.ts` — Uses `autogenerated` sidebars from directory structure
- `versions.json` — Currently tracks only `v0.24.0`

### Source Code Structure

- `website/src/components/` — React components (APITable, Blockquote, BrowserWindow, Details, DocCardItemIcon, Icon, NavbarItem, ScrollArea, Tag)
- `website/src/plugins/` — Custom Docusaurus plugins:
  - `blog-plugin.js` — Enhanced blog plugin with custom behavior
  - `tailwind-plugin.js` — Tailwind CSS integration
- `website/src/css/` — SCSS stylesheets (custom.scss, ragflow-variables.scss, infima-variables.scss)
- `website/src/theme/` — Custom Docusaurus theme components
- `website/src/utils/` — Prism syntax highlighting themes (prismLight, prismDark)

### Styling

- **Tailwind CSS** via custom plugin
- **Sass/SCSS** via `docusaurus-plugin-sass`
- Dark mode is forced (light mode disabled)
- Custom fonts: Poppins (headings) and Inter (body)

## Important Notes

- PRs to this repo are not merged — documentation contributions should go to the [main RAGFlow repo](https://github.com/infiniflow/ragflow/tree/main/docs)
- Edit URLs in `docusaurus.config.ts` point to the InfiniFlow/ragflow repo for PRs
- The `/docs` route automatically redirects to `/docs/dev` via client-redirects plugin
- Blog posts extract their first image for social cards in the frontmatter hook
